•	Traverse into the folder that has all the files using your command prompt/terminal and install the file “req.txt”

•	After installing all the requirements open the source code in visual code studio and open “app.py” and traverse into the folder again.

•	Now install sentence-transformers and streamlit==1.31.1 using pip command.

•	Now execute the command “streamlit run app.py”

•	Now you get a window popped up from your default browser. That’s it you have successfully implemented the tool in your environment.
